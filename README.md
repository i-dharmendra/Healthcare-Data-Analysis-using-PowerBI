## Overview
This project delivers a Power BI dashboard designed to help hospital administrators, analysts, and stakeholders explore healthcare metrics with interactive visuals such as KPIs, trend charts, and demographic breakdowns.

## Features
- Interactive visualizations and slicers  
- Key Performance Indicators (patient count, revenue, wait times)  
- Demographic analysis (age, gender)  
- Drill-downs and filters  
- Trend analysis across time  

## Data Description
**Datasets**  
- `patients.csv`: patient demographics and visit records  
- `hospital_financials.xlsx`: charges and costs  
- Additional reference tables (departments, staff, treatments)

## Architecture & Tools
- **Power BI Desktop** (.pbix file)  
- **Power Query** for data transform & ETL  
- **DAX** for calculated measures  
- Optional visuals from Marketplace
